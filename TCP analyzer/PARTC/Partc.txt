************************************************************************************************************************************************************************
overview of the code
 To get packets from pcap file I used dpt library , then etracted each packet in the pcap file using struct module, for the first question reassambled request and response 
 between each tcp connecction.
 For the second question estimated the connection type using number of tcp connections made.
 For the third question calculate packets, total time and raw data from pcap file.
